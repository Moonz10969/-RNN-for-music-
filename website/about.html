About this project
The mission of this project is to generate polyphnic music for background music. Using Artificial intelligence to benefits producer by providing extra choices effectively with less cost and time.
Google's polyphonic RNN model  was used for the machine learning model training. The model was trained with the data  collected from Freemidi.org and Classical Archives LLC to generate classical, pop, and rock music phrase.
Current State
In current state, our product provide 3 different genre (Classical, Pop, and rock) for you to listen and download.
Wav files are provided for creation purpose.
